# 일본 IT 기업 평가 스킬

## 사용자 프로필 (고정값 — 매번 재입력 불필요)

| 항목 | 내용 |
|------|------|
| 전공 | 인공지능소프트웨어학과 (한국 전문대, 2027년 2월 졸업예정) |
| 일본어 | JLPT N2 |
| 기술 스택 | Spring Boot, FastAPI, React, Gemini API, YOLO, Raspberry Pi |
| 포트폴리오 | AI 분실물 관리 시스템(캡스톤), 주식 뉴스 브리핑 서비스, AI 대화앱, 주식 분석 웹앱 |
| 취업 목표 | 일본 중소 SIer, 직접 고용(正社員), 비자 스폰서 필수 |
| 졸업 구분 | 전문학교 졸업(専門卒) 수준으로 처리될 가능성 높음 |

---

## 입력 방식

**A. 기업명만 제공된 경우 (웹 검색 모드)**
→ 아래 Step 1부터 웹 검색으로 데이터 수집 후 평가 진행

**B. PDF / 자료 제공된 경우 (수동 분석 모드)**
→ 제공된 자료 + 필요 시 보조 웹 검색 병행

---

## Step 1 — 데이터 수집 (웹 검색)

기업명이 주어지면 다음 항목을 검색으로 수집한다.
검색 키워드 예시: `{회사명} 新卒採用 2026`, `{회사명} 年収 平均`, `{회사名} 外国人 採用`, `{회사名} 口コミ OpenWork`

수집 목표 데이터:
- 사업 내용 / 주력 서비스 / 기술 스택
- 채용 직종 및 신입 모집 여부
- 평균 연봉 (OpenWork, 転職会議, 求人情報 등)
- 직접 고용 여부 (正社員 vs 派遣・SES 비율)
- 외국인 채용 실적 / 비자 지원 여부
- 오피스 위치 및 주변 생활 인프라
- OpenWork 등 구성원 리뷰 (워라밸, 이직률 등)
- **최근 프로젝트·사례** (IR자료, 공식 뉴스룸, プレスリリース, 導入事例 페이지 검색)

최근 프로젝트 검색 키워드 예시:
`{회사명} 導入事例`, `{회사명} プレスリリース 2025 2026`, `{회사명} 事例 AI`, `{회사명} IR ニュース`

---

## Step 2 — 6개 항목 평가

수집된 데이터를 바탕으로 아래 6개 항목을 각 **10점 만점**으로 평가한다.
점수는 추정치임을 명시하고, 근거를 1~2문장으로 서술한다.

### 📌 평가 기준 상세

#### 1. 전공 적합도 (10점)
AI·소프트웨어 개발 직무와의 연관성을 평가.
- 10점: AI/ML 개발, 자사 서비스 개발, FastAPI·Spring Boot 등 스택 활용 가능
- 7점: 일반 웹 개발, SI 프로젝트 (기술 성장 가능)
- 4점: 인프라·운용 중심, 개발보다 보수 업무 위주
- 1점: IT와 무관하거나 단순 사무 업무

#### 2. 장기 근무 가능성 (10점)
외국인이 안정적으로 장기 근속할 수 있는 환경인지 평가.
- 정사원 직접 고용 여부 (파견·SES면 대폭 감점)
- 비자 갱신 지원 실적
- 이직률 / 평균 근속 연수
- 외국인 재직자 유무

#### 3. 합격 가능성 (10점)
현재 프로필로 서류·면접 통과 가능성.
- 전문대졸(専門卒) 응시 가능 여부
- 외국인 신입 채용 전례
- JLPT N2 요구 수준과의 일치도
- 포트폴리오(AI 프로젝트)의 차별화 가치

#### 4. 경쟁률 (10점) ※ 낮을수록 유리 → 점수 높음
입사 난이도를 역산하여 점수화.
- 10점: 소규모(50인 이하), 외국인 환영, 마이나비 비주류
- 7점: 중소 규모(50~300인), 일반적 경쟁률
- 4점: 유명 중견, 채용 인원 적음
- 1점: 대기업·유명사, 경쟁 극심

#### 5. 연봉 수준 (10점)
신입 초임 기준 평가 (도쿄 기준 중앙값 약 23~25만엔/월).
- 10점: 월 27만엔 이상 또는 연 350만엔 이상
- 7점: 월 23~27만엔 (연 280~350만엔)
- 4점: 월 20~23만엔 (생활 가능하나 빠듯)
- 1점: 월 20만엔 미만 또는 정보 없음

#### 6. 주변 인프라 (10점)
오피스 위치 기준 생활 편의성 평가.
- 대중교통 접근성 (도심 vs 외곽)
- 주변 물가 / 주거비 수준
- 외국인 거주 환경 (한국인 커뮤니티, 다문화 편의시설)
- 재택근무 / 플렉스타임 제도 여부

## Step 3 — 결과 출력 형식

아래 형식으로 한국어로 작성한다. 간결하고 가독성 있게.

---

### 🏢 [기업명] 평가 리포트

**기업 한 줄 요약**: (사업 내용, 규모, 위치를 1문장으로)

---

**📊 종합 평점: XX / 60점** → ⭐ [등급 라벨]

| 항목 | 점수 | 한 줄 근거 |
|------|------|-----------|
| 전공 적합도 | X/10 | ... |
| 장기 근무 가능성 | X/10 | ... |
| 합격 가능성 | X/10 | ... |
| 경쟁률 (낮을수록 유리) | X/10 | ... |
| 연봉 수준 | X/10 | ... |
| 주변 인프라 | X/10 | ... |

---

**✅ 지원 추천 여부**: [강력 추천 / 추천 / 보통 / 비추천]

**💡 핵심 장점** (2~3가지 bullet)

**⚠️ 주의사항 / 리스크** (1~2가지 bullet)

**🚀 최근 주요 프로젝트 / 사례** (면접 활용용)
검색으로 확인된 최근 1~2년 이내 프로젝트·도입 사례·발표 자료를 나열한다.
각 항목은 아래 형식으로 작성:
- **[연도] 프로젝트명 또는 파트너사** — 내용 1~2문장. *→ 면접 활용 포인트: 이 사례를 언급하며 "~에 기여하고 싶다"는 식으로 연결 가능*

정보가 없으면: "공개된 최근 프로젝트 사례를 찾지 못했습니다. 면접 시 '最近注力されているプロジェクトを教えていただけますか？'로 직접 질문 권장."

**📝 지원 전 확인 사항** (면접·ES 전 체크해야 할 것)

## 주의사항

- 연봉·경쟁률 데이터가 없으면 "정보 없음 (추정 보통 수준)"으로 표기하고 감점하지 않는다
- 파견·SES 비율이 높은 회사는 장기 근무 가능성 항목에서 최대 -3점 적용
- 비자 스폰서 여부가 불명확하면 합격 가능성 항목에서 -1점 처리
- 점수는 어디까지나 공개 정보 기반 추정임을 리포트 하단에 명시할 것
